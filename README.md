# Coffee Website

A clean, fast coffee shop UI built with plain **HTML / CSS / JavaScript**.  
Includes **menu categories**, a **testimonials slider**, a **gallery**, and a **contact form** UI.

## Features

- Responsive layout
- Menu categories (hot beverages, cold beverages, desserts, special combo)
- Gallery section
- Testimonials slider
- Contact form UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
